## 📂 Featured Projects

### 🎓 Sohayika — Exam Preparation Platform
> `Laravel 12` · `PHP 8.2` · `MySQL` · `React` · `REST APIs` · `Queues` · `OAuth`

A Bengali exam preparation platform with online exams, a question bank, and performance analytics.

- Backend features and REST APIs for exam delivery, submissions, and result computation
- Queue-based background processing and social/OAuth authentication
- User account management, with a focus on reliability and performance

🔗 **[Live](https://sohayika.com)**

---

### 🏥 Niramoy — Healthcare Appointment Platform
> `Next.js` · `Express.js` · `MongoDB` · `Firebase` · `JWT` · `Stripe` · `Recharts`

A full-stack platform where patients book appointments with verified doctors.

- Role-based dashboards for patients, doctors, and admins
- Secure authentication with Firebase and JWT, plus protected API routes
- Stripe payment integration with double-entry payment records in MongoDB
- Advanced doctor search, sorting, pagination, and an admin analytics dashboard

🔗 **[Live](https://niramoy-client.vercel.app)**
